Module Hierarchy

  • CustomTreeCtrl: Description =========== CustomTreeCtrl is a class that mimics the behaviour of wx.TreeCtrl, with almost the same base functionalities plus some more enhancements.

  • CustomTreeCtrl.GenericTreeItem: This class holds all the information and methods for every single item in CustomTreeCtrl.
  • CustomTreeCtrl.TreeItemAttr: Creates the item attributes (text colour, background colour and font).
